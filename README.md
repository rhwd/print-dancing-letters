# Dancing Letters

This Rust project allows you to print phrases onto dancing letters GIFs.

## Dependencies

  You'll need [ffmpeg](https://ffmpeg.org/) installed.

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/rhwd/print-dancing-letters.git

2. Build the project:

   ```sh
   cargo build

3. And run!:

     ```sh
   cargo run -- hello
